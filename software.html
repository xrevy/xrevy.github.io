Music Vizualizer

SS 2019

This university project represents a music visualizer and was made in java based processing.

This program takes a music file as input and analyzes its frequencies. Based on the values of bass, mid and high frequencies, symbols and geometry are formed.
The main visualization is a big triangle that moves to the frequencies. Each edge of the triangle represents either bass, mid or high frequency values. In the background there are different things to see. Depending on the chosen display modus, the user can see pyramids, spheres, rectangle planes or arches in the background. These "particles" are growing based of the sum of the frequencies, the louder an audio frame, the bigger the particles.
The user can choose between non-copyrighted preset songs or can even upload their own song. If a song has ended, the next song will be played.
The user can also choose the displaying mode and if they want to skip the song.

Self-Sorting Lists and Sorting Algorithms

SS 2019

While learning how to code in Java we had to do weekly assignments in this course. Part of that assignments were tasks like self-sorting lists and sorting algorithms.
To use self-sorting lists we had to implement different methods to actually self-sort list elements. These were Naive (no actual sorting, new elements just are inserted at the beginning of the list), MF-Rule ("Move-to-Front", each element that is already in the list and should be inserted in the list, just gets moved to the front of the list, else like naive), T-Rule ("Transpose", each element that is already in the list and should be inserted in the list gets swapped with its predecessor, else live naive) and FC-Rule ("Frequency Count", each element in the list is sorted after its frequency, how often it is inserted to the list. The more an element is tried to be inserted, the further to the front of the list this element is sorted). Apart from that we had to know how many sorting algorithms would work, like bubble sort, insertion sort and quick sort for example. We've practically implemented the merge sort algorithm as an assignment.
